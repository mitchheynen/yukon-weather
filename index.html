Deployment notes

This file is designed for GitHub Pages, Cloudflare Pages, or Netlify. No backend is required.

How it works: the dashboard uses official Open-Meteo endpoints for generic forecast, GEM, ECMWF, and GFS. It then computes a simple consensus average and confidence score in the browser.

Recommended use: push this file to a GitHub repository as index.html, enable GitHub Pages, and use the default free subdomain.
